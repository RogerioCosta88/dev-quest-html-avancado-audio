Audio

Usamos a Tag audio para adicionar audios nos sites

Controls

Controls é um atributo da Tag audio. O controls adiciona os botões para controle do audio, como: play, pause, avançar e retroceder o som.

Tittle

Já o atributo tittle adiciona um título ao arquivo de audio, da mesma forma que acontece na Tag img.

Autoplay

O autoplay é um atributo que dá play no audio assim que a página carrega, sem que o usuário precise clicar no botão de play. O navegador Chrome não permite que esse atributo funcione, sendo necessário usar uma biblioteca de JavaScript para que o atributo autoplay funcione no Chrome.

Preload

Preload é um atributo usado para carregar o audio e algumas informações assim que a página carregar. O valor padrão do atributo preload é o auto que carrega o audio e as informações automaticamente, sem que o usuário precise no clicar no botão de play. Mas podemos escolher entre outros dois valores que são melhores para que o usuário gaste menos dados: metadata que carrega somente algumas informações, como: o tempo do áudio, ou none que só irá carregar tanto o áudio, quanto as informações do áudio, quando o usuário clicar no botão de play.

Extensões

Alguns navegadores não aceitam certos tipos de extensões de áudio, exemplo: o navegador Safari não recenhece a extensão .ogg, para evitar esse tipo de problema, é interessante colocar mais de uma opção de extensão na página usando a Tag source dentro da Tag audio, usamos uma Tag source para cada extensão. Adicionando o caminho do áudio no atributo src para cada extensão e também um atributo type.
